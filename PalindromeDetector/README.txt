Palindrome Detector

A palindrome is a phrase that reads the same forwards as it does backwards.
For example, “a man, a plan, a canal, Panama,” is a palindrome.
This is a program that uses a stack to check for palindromes in each line of a text file.
Program outputs the palindromes that it finds in the document. For example:

"a man, a plan, a canal, Panama" is a palindrome.
"Don’t nod" is a palindrome. 
"Taco Cat!" is a palindrome.
